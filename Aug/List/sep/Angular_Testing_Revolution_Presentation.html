üéØ Vitest Key Features

1. Polling Feature

Automatically waits for async operations

await expect.poll(() => value).toBe(3)

2. Soft Assertions

Run all tests even if one fails

expect.soft(value).toBe(true)

3. Test Context

Dependency injection for tests

test('name', async ({ context }) => {})

4. Browser Mode

Test in real browser environment

vitest --browser

5. Fake Timers

Control time in tests for async operations

vitest.runAllTimers() // Fast-forward time

What is Testronaut?

Community-driven Playwright Component Testing runner that lets you test Angular components in isolation using real browser automation

Why Component Testing?

  • Test components in isolation
  • No full app startup needed
  • Faster than E2E tests
  • Real browser environment
  • Visual feedback

Advantages

  • üéØ Focused testing
